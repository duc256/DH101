# Accessibility 

This site was built by Duc Tran for DH101 as a course portfolio. It includes my weekly Makes and Reflections from Weeks 1 through 8, plus project pages where I share links, screenshots, maps, and network visualizations. Most pages are written in Markdown and include images or embedded media to document my process and outcomes.

I have tried to make this site accessible by:
- writing pages with clear headings and simple structure so they are easier to skim and navigate
- adding alt text for images and GIFs so the content is understandable with screen readers
- using descriptive link text that explains where a link goes, instead of vague phrases like “click here”
- keeping visual information supported by text, so key points are not only communicated through color or layout

Known limitations:
- Some embedded tools such as Kumu maps or Google My Maps may have accessibility limits that I cannot fully control, since they are external platforms. When possible, I provide a direct link and a screenshot with alt text as a backup.

If you encounter barriers using this site, please contact me at tran_d3@denison.edu.
